# \<voldemort-element>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i voldemort-element
```

## Usage
```html
<script type="module">
  import 'voldemort-element/voldemort-element.js';
</script>

<voldemort-element></voldemort-element>
```
